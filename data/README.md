# Telescope catalog provenance

`Telescope_Site_Matrix.csv` is a snapshot of the ngEHT working telescope
catalog maintained in `ngeht-weather/regenerate/Telescope_Site_Matrix.csv`.
The catalog originated in the MIT-licensed
[`Smithsonian/ngeht-sims`](https://github.com/Smithsonian/ngeht-sims)
repository.

- Snapshot imported: 2026-07-05

Run `npm run import:sites` after replacing the CSV snapshot. Domain-specific
display names and visual-group defaults are intentionally maintained in
`site-overrides.json`.
